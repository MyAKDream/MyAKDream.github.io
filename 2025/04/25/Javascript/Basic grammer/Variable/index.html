变量和常量

概述

变量可以重新赋值，而常量不可以

思考

  • const 声明的数组可以添加或删除吗
  • const 声明的对象可以添加或者修改吗

可以的，因为数组和对象在JS中属于引用类型，对其做添加、删除等操作，并不改变内存地址。本质上就是修改指针指向的空间。
