操作系统的发展和分类

说明

  • 如果只想了解操作系统应用：
    • 请忽略这一节
  • 如果想深入学习操作系统：
    • 熟悉手工操作阶段，批处理阶段，分时操作系统，实时操作系统的特点。
    • 其他的了解。

手工操作阶段（此阶段无操作系统）

用户在计算机上进行计算全部都需要人工干预，如程序输入，运行，结果的输出等。

缺点：

  • 用户独占全机，虽然不会出现资源被占用的现象而等待，但是资源利用率低。
  • CPU等待手工操作，CPU利用不充分

批处理阶段

单道批处理系统

将一批作业一脱机的方式输入磁带，通过监督程序，使这批作业的处理是成批进行的。

主要特征：

  • 自动性：磁带上的作业依次运行，不需要人工干预
  • 顺序性：磁带的各道作业顺序进入内存，先调用的作业先完成
  • 单道性：内存中只能由一道程序在运行

多道批处理系统

将用户的作业都放到外存上形成一个队列，作业调度程序按一定的算法从后备队列中选择若干作业导入内存。当某程序因I/O操作而停止运行时，CPU会立刻执行另一道程序。

分时操作系统

计算机以时间片为单位轮流为各个用户/作业服务，各个用户可通过终端与计算机交互。

主要优点：用户请求可以被即时响应，解决了人机交互问题。允许多个用户同时使用一台计算机，并且用户对计算机的操作相互独立，感受不到别人的存在。
主要缺点：不能优先处理紧急任务。

实时操作系统

主要优点：能够有限响应一些紧急任务，某些任务不需要时间片排队。

在实时操作系统的控制下，计算机系统接收到外部信号后即时处理，并且需要严格的时限内处理完事件。实时操作系统的主要特点是及时性和实时性。

网络操作系统

是伴随着计算机网络的发展而诞生的，能把网络中各个计算机有机地结合起来，实现数据传送等功能，实现网络中各种资源的共享（如文件共享）和各台计算机之间的通信。（如：Windows NT就是一种典型的网络操作系统，网站服务器就可以使用）

分布式操作系统

主要特点是分布性和并行性。系统中的各台计算机地位相同，任何工作都可以分布在这些计算机上，由它们并行、协同完成这个任务。

个人计算机操作系统

如Windows XP、MacOS，方便个人使用
