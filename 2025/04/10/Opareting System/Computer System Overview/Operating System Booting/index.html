操作系统引导

概述

我们新买来1T的硬盘，插到电脑中，此时电脑不能正常开机启动，需要我们安装操作系统。

安装操作系统之后硬盘被分为了几个盘，其中C盘式系统盘，操作系统程序存放在这个盘中。

操作系统引导过程

操作系统的引导过程本质就是电脑开机的过程。

操作系统引导：

  • CPU从一个特定主存地址开始，取指令，执行ROM中的引导程序。（先进行硬件自检，再开机）
  • 将磁盘的第一块——主引导记录读入内存，执行磁盘引导程序，扫描分区表。
  • 从活动分区（又称主分区，即安装了操作系统的分区）读入分区引导记录，执行其中程序。
  • 从根目录下找到完整的操作系统初始化程序（即启动管理器）并执行，完成开机的一系列操作。
